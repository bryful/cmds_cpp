# r2z

RARファイルをZIPファイルに変換するWindows用コマンドラインツールです。

## 機能
- 指定したRARファイルをZIP形式に変換します。
- アーカイブ内に`.scr`ファイルが含まれている場合、自動的に除外します。
- 7-Zip（7z.exe）を利用して展開・圧縮を行います。

## 使い方
1. 7-Zip（7z.exe）が `C:\Program Files\7-Zip\7z.exe` にインストールされていることを確認してください。
   - 別の場所にある場合は、実行ファイルと同じフォルダに `.pref` ファイルを作成し、1行目に7z.exeのパスを記述してください。
2. コマンドプロンプトで以下のように実行します。

```
r2z.exe <RARファイル1> <RARファイル2> ...
```

例:
```
r2z.exe sample1.rar sample2.rar
```

- 変換後、同じフォルダにZIPファイルが作成されます。
- `.scr`ファイルが含まれている場合は警告が表示され、ZIPには含まれません。

## ビルド方法
Visual Studio 2019以降でプロジェクトを開き、ビルドしてください。
